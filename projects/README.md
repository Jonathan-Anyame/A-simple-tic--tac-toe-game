# Beginner C++ Mini Projects

This folder contains three beginner-friendly C++ mini projects with simple build and run instructions.

Projects:

- `guess-number/` - Guess the Number game.
- `calculator/` - Simple REPL calculator.
- `rock-paper-scissors/` - Rock-Paper-Scissors game vs computer.

Building (example using clang++):

cd projects/guess-number
clang++ main.cpp -o guess-number
./guess-number

Repeat for other projects by entering their directories.
